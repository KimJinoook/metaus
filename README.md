# 📌 프로젝트 개요

- 프로젝트 명 : Metaus
- 프로젝트 기간 : 2022.05.30 ~ 2022.07.18
- 사이트 개요   
> 4차산업의 발전으로 메타버스라는 새로운 시장이 확장하며 구찌, 현대차등 다양한 대기업이 가상세계에 입점 및 상품을 판매하고 있습니다.   
> 이러한 가운데 기술이 없어 진출하지 못하는 소상공인들의 메타버스 진출을 위해 '크리에이터' 와 '소상공인'을 연결하는 플랫폼으로 메타어스를 제작했습니다.
